# Art Rules

Use this group when changed work affects art direction, composition, generated
art, visual polish, material quality, screenshots as final output, or a visual
target.

## Order

1. Start with [COMMON.md](COMMON.md).
2. Add numbered checks only when their "Use When" section matches the task.

## Checks

### [QART_COMMON - Art Common](COMMON.md)
Checks obvious art/presentation blockers: placeholder-looking output, unclear
subject/action, broken or stretched assets, target-style mismatch, or no
actual-output proof.

Use first for any art, generated image, material, screenshot, or final-looking
player output change.

### [QART_001 - Art Direction Fit](checks/QART_001_art_direction_fit.md)
Checks whether the output fits the accepted visual target, composition,
material feel, polish bar, and current game's style.

Use when generated art, composition, material feel, visual polish, final
presentation screenshots, or accepted visual target changed.

Record applied checks in the task log as `Quality: QART_001=pass` or
`Quality: QART_001=block`.
