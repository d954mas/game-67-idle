# Asset Workflow Art Jobs

Art job tools keep generated-art work explicit enough for agents to resume,
review, validate, and trace provenance.

Use this group when creating or validating a generated-art task, source-family
prompt packet, generation record, or project asset-boundary audit.

This group does not generate images by itself, does not store accepted assets,
and does not build engine integration artifacts. Accepted assets should move through
`ai_studio/assets/storage/` and file preparation through
`ai_studio/assets/prep/`.
