# Облачная видео-генерация: API и подписки (2026-07-05)

# СВОДКА (для лида)

**Про Gemini/Veo (прямой ответ):** Да, у Google есть видео-генерация — **Veo 3.1** через Gemini API и в подписках Google AI. Технически умеет всё нужное (image-to-video, «Frames to Video» = первый+последний кадр, коммерция разрешена на платном API, видимого вотермарка на API нет — только невидимый SynthID). **НО для нашей задачи это худший выбор:** (1) кинематографичная/фотореалистичная модель — «подсвечивает», добавляет глубину/DoF/звук, ломает плоский 2D-стиль и ровный зелёный фон под хромакей; (2) минимум клипа 4 сек, дешёвый тариф Veo 3.1 Fast = $0.15/с × 4с = **$0.60/клип** (Standard 4с = $1.60) → ~$18 за персонажа; (3) подписки (AI Pro $19.99 ≈ 3 Veo Lite/день; Ultra $200 ≈ 5/день) — дневные лимиты, не для батч-итераций. **Вердикт: нет.**

**Топ-2 дешёвых варианта (под 2с/зелёный/плоский стиль):**
1. **Hosted WAN 2.2 на fal.ai** — та же модель, что локально. Посекундная оплата (16 fps, БЕЗ минимума 5с): **480p = $0.04/с → ~$0.08 за 2с; 720p = $0.08/с → ~$0.16 за 2с.** Нативный FLF (end-image), Apache-2.0 = коммерция без вотермарка, ноль отличий стиля/зелёного. Источник: страница модели fal.ai (первичная).
2. **Seedance 1.0 Lite** — $0.18 за 720p/5с (2.0 Fast ~$0.09-0.11/с), FLF в 1.5+, коммерция ок. Но ДРУГАЯ модель → риск дрейфа стиля/зелёного, минимум 4-5с.

**Гибрид = победитель:** локальные черновики (бесплатно, 35с) + облачные финалы на hosted WAN 720/1080p = **~$0.45-0.50 за персонажа** (3 финала) при нулевой дельте стиля — та же модель. Снимает 1080p-нагрузку с 12GB.

**Рекомендация:** локальный WAN остаётся дефолтом. Единственный кандидат на платный пилот — **fal.ai (WAN 2.2 i2v + FLF2V)**: ~10-20 финалов 720-1080p, бюджет <$5, сравнить с локальными 480p. Veo/Kling/Runway/Pika-подписки под задачу НЕ брать (минимумы 4-5с, вотермарк/коммерция за верхними тарифами, фотореал-байас).

# Три сценария (персонаж: 3 состояния ≈ 27 черновиков + 3 финала)

- **(a) Всё локально:** $0, ~18-20 мин GPU, безлимит, FLF нативно. Дефолт.
- **(b) Гибрид (РЕКОМЕНДАЦИЯ):** черновики локально $0 + 3 финала hosted WAN 720p = $0.48 (или 1080p WAN 2.6 = $0.42). **~$0.45-0.50/персонаж**, ноль дельты стиля, matting без перенастройки.
- **(c) Всё облако (hosted WAN):** 27×$0.08 + 3×$0.16 = **~$2.64/персонаж**. Без вотермарка, коммерция, FLF; параллелится; но очереди и рост с итерациями.

Для сравнения: отвергнутый мир «$1/клип» = $30/персонаж. Hosted WAN ~11× дешевле, гибрид ~60×.

# Почему hosted WAN — единственный убедительный cloud-комплемент

1. Идентичность модели = ноль «фотореализации» и идентичный зелёный → chroma-matting без перенастройки.
2. Посекундная оплата без минимума 5с.
3. Нативный FLF (fal end-image + WAN FLF2V) — под раскадровку A→B и лупы.
4. Apache-2.0 — коммерция без вотермарка.
5. Даёт 720p/1080p финалы, которые 12GB тянет плохо.

Провайдеры: fal.ai (зрелый, i2v $0.04-0.08/с), WaveSpeed (выделенный WAN FLF2V, но 2.1: $0.15-0.40/5с), Replicate. Латентность: 2с@480p ~10-30с + очередь ≈ локальные 35-54с, но параллелизуемо.

# Подписки $8-15/мес — что реально покупают

- Pika Standard $8 — вотермарк + БЕЗ коммерции (Pro $28).
- Runway Standard $12 = 625 кр = 125с Gen-4 Turbo (~25 клипов 5с), коммерция на платном — лучший из подписочных, но чужой стиль.
- Kling $10-15 ≈ 660 кр (~44 клипа 5с), минимум 5с.
- PixVerse $9.99 — 120 кр/5с v5.5, аниме-режим; free = вотермарк.

Вывод: под батч-итерации per-second API (WAN) бьёт кредитные подписки.

# Риски / оговорки

- Агрегаторные блоги противоречивы (напр. «Veo $0.03/с» — вероятно Lite/промо); официально Veo Fast $0.15/с, Standard $0.40/с. Самая надёжная цифра — fal WAN 2.2 (первичная страница).
- Версии быстро устаревают (WAN 2.6/2.7, Kling 3.0, Seedance 2.0, Veo 3.1 на дату исследования) — перед пилотом сверить.

- fal/Replicate: материалы принадлежат пользователю; TOS агрегатора бегло проверить перед продакшеном.
