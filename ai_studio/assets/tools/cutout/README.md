# Asset Tools Cutout

Cutout tools turn keyed or paired source images into clean RGBA assets.

This group owns alpha extraction, key-matte cleanup, dual-plate extraction, trim
support, transparent-edge hygiene, and reports for cutout quality. It does not
own crop planning, storage manifests, engine integration, or browser UI.

Use this group when source art has a chroma key, a white/black dual-plate pair,
dirty transparent RGB, or alpha edges that must be cleaned before the asset is
accepted or used in a project.
