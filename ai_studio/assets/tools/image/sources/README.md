# image/sources — source image conventions

Working conventions for source images before they become accepted project
assets.

Draft and generated images should be copied to `tmp/assets/` while iterating, so
they are not lost in the default generator cache. Accepted assets still move to
the owning game, template, or shared asset location with provenance and license
metadata.

This folder does not own image generation. Generation remains skill/workflow
owned; this stage starts once a source image file exists.
