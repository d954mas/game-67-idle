# Manual Asset Prep

Browser surface for local, manual source-image preparation.

This page owns interactive image loading, grid/tile slicing, simple alpha
transforms, preview overlays, crop-plan JSON export, and PNG/ZIP downloads. It
does not write into backlog storage, does not claim license/provenance, and does
not replace the scripted asset tools for accepted production evidence.

Use this when a source sheet needs fast manual inspection or one-off prepared
outputs before accepted files are recorded through the normal asset pipeline.
