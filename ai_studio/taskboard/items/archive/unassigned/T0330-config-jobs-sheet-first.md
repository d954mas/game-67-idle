## What

Единственный недостающий кубик после ресёрча Declarative Art (доклад TiltShift,
https://www.youtube.com/watch?v=y9znnFAQWOQ, 2026-04): `gen_batch.py` умеет есть
`jobs.json`, но никто не генерит его из декларативного конфига. Экспандер:

- Конфиг пака: style prefix (verbatim style-блок из style card / T0208 /
  art_contract) + object suffix (тип объекта, шаблон [SUBJECT]) + wildcard-оси
  (материал/форма/**грейд** и т.п.) + anchor ref.
- Sheet-first, не 1-промпт-на-ассет: крупные оси (грейд, материал) фиксируются
  на лист, мелкая ось варьируется по ячейкам листа (2x2..3x3); джоба несёт
  cell-манифест — какая ячейка = какие значения осей. Манифест дальше именует
  нарезку (intake/crop) и уходит в canvas meta каждого вырезанного элемента
  (config ref + координаты осей) — отбор в канвасе фильтруем по осям.
- Anchor-правило: принятый hero-ассет пака подставляется `--input-image` во все
  сиблинг-джобы автоматически (флаг уже работает на обоих путях; помнить про
  разницу: REST = /images/edits, codex = inline).
- Кап кардинальности осей обязателен: у нас 30-60 с/картинка и платно —
  экономика «1000 на конфиг» из доклада не переносится (их держат локальные
  бесплатные сэмплеры).
- Seed-фичи не делать: gpt-image seed не принимает (миф закрыт в
  ai_studio/assets/canvas/docs/research_genart_workflow_2026-07-03.md).
  Замена воспроизводимости = овергенерация N на ячейку + гейт/лид.
