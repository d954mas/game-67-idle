## Goal

Make AI Studio able to ship repeated small commercial web games through a measurable release loop: package, publish target, analytics, and kill/continue decision.

## In scope

- Publish target adapters for itch, self-hosted, and portal-ready web builds.
- Analytics scorecard over existing game analytics/event exports.
- Release gates that turn a copied template game into a browser-tested artifact.
- Task slices for portfolio iteration and monetization evidence.

## Out of scope

- Platform account setup, live payouts, and external credential storage.
- Full portal SDK integration until adapters and smoke checks prove the local package.
- Game-specific content production for rb-dark-rpg or another title.
