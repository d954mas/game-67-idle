## Current Reference Digest

Mode: `central deconstruction`.

Sources checked:

- Melvor Steam page;
- live Melvor site;
- Melvor Skills wiki;
- Melvor v0.01-v0.08 changelog pages;
- Melvor v1.0 changelog page.

Observed facts:

- Melvor v0.01 launched with Woodcutting as the only real skill.
- v0.02 deepened Woodcutting with prestige, nests, stats, random-event UX, and
  balancing before broad expansion.
- v0.03 added Fishing and Firemaking, including a resource dependency from logs
  into Firemaking.
- v0.04 redesigned the project and added Cooking, Mining, and Smithing.
- v0.05 added Combat after resource and gear systems existed.
