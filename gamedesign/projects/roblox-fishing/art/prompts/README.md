# Splash Rods Prompt Packets

These prompt packets are for future reusable runtime UI source families. They
are not generated yet and do not close final-art gates.

- `splash-rods-blank-ui-kit-v1-prompt.md/json`: slice9-safe blank panels,
  buttons, bars, and frames with no baked text.
- `splash-rods-icons-v1-prompt.md/json`: isolated semantic icons for coins,
  backpack, fish index, rods, bait, shop, rarity, and locked states.
- `splash-rods-ui-decor-v1-prompt.md/json`: separate non-stretch overlays such
  as corner caps, badges, glows, locks, and plaques.
