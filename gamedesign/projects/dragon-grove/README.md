# Dragon Grove

Project wiki for the active prototype `dragon-grove`.

## Concept

Original merge-3 dragon grove puzzle: merge eggs, sprouts, and dragons on a
small Y-up grid to restore a clearing.

## First Slice

- One native PC scene.
- One merge-ready 5x5 grove grid.
- One visible merge action and reward/restore response.
- One blocked/no-merge state.
- Product/readability proof before expanding content.
