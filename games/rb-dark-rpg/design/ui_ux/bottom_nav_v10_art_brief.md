# Bottom Nav V10 Art Brief

Date: 2026-07-04

Status: candidate for artlead review, not production-sliced.

## Change From V9

V9 made `Место` a magnifying glass, but the lens read like an opaque brass disk.
V10 keeps the same UX decision and fixes the material read:

- transparent amber glass lens;
- visible crescent highlight;
- stone tile visible through the glass;
- bronze rim only around the edge;
- no frying-pan / coin / shield read.

## Keep

- Equal outer button size and equal runtime hitboxes.
- Empty lower trough reserved for runtime text.
- No baked text in the PNG.
- Muted dark fantasy palette.
- `Еще` as three physical brass dots.
