# RB Dark RPG Knowledge Sources

Store source notes and reference packets that support this game's private
knowledge base.

For each source, record:

- title, link or local path, and checked date;
- source quality;
- what it proves and what it does not prove;
- evidence labels: `observed`, `secondary`, `inferred`, or `unknown`.

Promote only game-specific conclusions into `../index.md` or focused knowledge
pages. Reusable cross-game rules belong in the shared game-design knowledge base.
