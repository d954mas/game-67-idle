# Private Game Knowledge Base

Private knowledge base for **web-dressup**.

Use it for accepted reference lessons, playtest findings, build observations,
and design decisions that should survive across sessions.

Do not put reusable cross-game rules here. Promote those to the shared
game-design knowledge base after rewriting them as general rules.

## Files

- `index.md` — map of durable game-specific knowledge.
- `sources/` — source notes, reference packets, and evidence for this game.
