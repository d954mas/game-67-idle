# Kenney Runway Audio

CC0 seed audio for Runway Awakening's game-owned cue catalog.

- `audio/sfx/ui_click.wav` comes from Kenney UI Audio, archive member
  `Audio/click1.ogg`. It drives the UI click and the brighter magic-flash cue.
- `audio/sfx/awakening_jingle.mp3` comes from Kenney Music Jingles, archive
  member `Audio/8-Bit jingles/jingles_NES00.ogg`. It is the temporary charge
  and reveal/victory sound for the MVP.
- Runtime resource IDs omit extensions so a later audio repaint can change
  delivery codecs without changing game code.
