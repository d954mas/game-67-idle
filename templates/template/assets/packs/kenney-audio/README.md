# Kenney Starter Audio

CC0 reference audio for the template's game-owned audio catalog.

- `audio/sfx/ui_click.wav` comes from Kenney UI Audio, archive member
  `Audio/click1.ogg`.
- `audio/music/demo_jingle.wav` comes from Kenney Music Jingles, archive member
  `Audio/8-Bit jingles/jingles_NES00.ogg`.
- Both committed files are mono 44.1 kHz signed 16-bit PCM WAV produced with
  the exact ffmpeg commands recorded in `assets.jsonl`.
- The original OGG files are source provenance only. The v1 runtime does not
  claim OGG support.
