# Kenney Starter Audio

CC0 reference audio for the template's game-owned audio catalog.

- `audio/sfx/ui_click.wav` comes from Kenney UI Audio, archive member
  `Audio/click1.ogg`.
- `audio/music/demo_jingle.mp3` comes from Kenney Music Jingles, archive member
  `Audio/8-Bit jingles/jingles_NES00.ogg`.
- The UI cue is mono 44.1 kHz signed 16-bit PCM WAV. The music fixture is mono
  44.1 kHz MP3 at 128 kbit/s. Exact ffmpeg commands are in `assets.jsonl`.
- The original OGG files are source provenance only. Runtime OGG support is not
  claimed; MP3 is the single compressed native/web delivery format in v1.
