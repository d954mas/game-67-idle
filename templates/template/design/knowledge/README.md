# Private Game Knowledge Base

This is the private game knowledge base for this game.

Use it for game-specific knowledge that should survive across sessions:
accepted reference lessons, deconstruction notes, playtest findings, build
observations, design decisions, and implementation-facing facts.

Do not put reusable cross-game rules here. Promote those to the shared
game-design knowledge base after they are rewritten as general rules.

## Files

- `index.md` - map of durable game-specific knowledge.
- `sources/` - source notes, reference packets, and evidence for this game.

Keep work status and TODO tracking in Taskboard, not in this knowledge base.
