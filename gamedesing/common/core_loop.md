# Core Loop

Тезис: игрок растет через короткие действия, а не через чтение правил.

## Минута Игры

1. Сделать `67-жест`.
2. Собрать мем-коины.
3. Запустить `Дело`.
4. Купить улучшение.
5. Увидеть рост героя, дома или `X/67`.

## Что На Что Влияет

- `67-жест` -> быстрый доход и эмоция.
- `Дела` -> idle-доход.
- `Навыки` -> доступ к новым делам.
- `Дом` -> визуальный рост и мягкий прогресс.
- `X/67` -> главный статус.
