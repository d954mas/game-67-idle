# Playtest Gates

Тезис: каждое решение должно проверяться поведением игрока.

## P0 Gates

- Игрок понимает цель `15/67` без объяснений.
- Первый рост статуса происходит за 45-90 секунд.
- Любой финальный выбор ведет к `15/67`.
- Ребенок может играть 5 минут с одним большим action.
- Родителю понятно, что `67` - мемный ранг, не опасный смысл.

## Инструменты

- balance simulator;
- GDD validator;
- desktop screenshot;
- mobile portrait screenshot;
- input emulation.
