# My Design Rules

Личные правила, которые уже считаем полезными.

## Правила

- Цель должна читаться за 5 секунд: `1/67 -> 15/67`.
- Главное число видно всегда. Связь: [[game_design_bible#Что Такое 67]]
- Мем должен быть действием, не только картинкой. Связь: [[game_design_bible#67-Жест]]
- Для ребенка: одна карточка = один следующий шаг.
- Если ресурс не влияет на выбор, его не показываем рано.
- Любой GDD-тезис должен вести к проверке: симулятор, screenshot, playtest gate.

## Не Делать

- Не складывать сюда длинные спеки.
- Не писать “можно попробовать” без причины и проверки.
- Не плодить валюты раньше, чем игрок понял core loop.
