# Child Safe Copy

Тезис: текст должен быть мемным, но мягким.

## Использовать

- `Дела`
- `помочь`
- `команда 67`
- `ночник 06:00`
- `Дворовая база`
- `Уютный угол 67`

## Не Использовать В UI

- тяжелую бедность;
- унижение;
- взрослый труд;
- страх;
- жесткий конфликт.

## Правило

Шутка должна поднимать героя, а не давить на него.
