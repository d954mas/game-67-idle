# P0 UI Copy

Статус: v0.1.
Назначение: единый player-facing copy source для первого web/mobile прототипа.

## Copy Rules

- Главное действие всегда `Сделать 67`.
- Internal `jobs` всегда показываются как `Дела`.
- На карточке показывать одну следующую причину/цель.
- Любая награда должна иметь видимый результат из `visibleResult`.
- Forbidden wording lives in `content_matrix.md`; this copy file contains only approved UI text.
